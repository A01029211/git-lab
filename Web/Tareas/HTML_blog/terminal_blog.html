Comandos

Propositos

cd

Cambia el directorio
cd .. Regresa al directorio anterior
ls Muestra todo el contenido dentro de ese directorio
pwd Imprime en qué directorio estás actualmente
clear Borra toda tu pantalla de los comandos previos
touch crea un nuevo archiv en el directorio en el que estes
mkdir Crea una neva carpeta dentro de ese directorio
cat este comando mas el nombre del archivo te muestra como se ve el contenido del archivo sin tener que abrirlo
vim Permite escribir dentro de un archivo sin salir de la terminal
rm Elimina un archivo
rm -r Elimina una carpeta
history Muestra el historial de comandos utilizados
cp Copia un archivo y lo mueve al directorio que quieras
