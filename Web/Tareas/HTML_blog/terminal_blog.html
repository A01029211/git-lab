Comandos en terminal

Comandos

Propositos
cd cambia el directorio
cd .. Se regresa al directorio anterior
ls Te muestra todo el contenido dentro de ese directorio
pwd Imprime en que directorio estas actualmente
clear Borra toda tu pantalla de los comandos previos
touch crea un nuevo archiv en el directorio en el que estes
mkdir Crea una neva carpeta dentro de ese directorio
cat este comando mas el nombre del archivo te muestra como se ve el contenido del archivo sin tener que abrirlo
vim El vim se usa como para escribir dentro de un archivo sin necesidad de salirte de la terminal
rm Eimina un archivo
rm -r elimina una carpeta
history
cp Copia un archivo y lo mueva al directorio que quieras
