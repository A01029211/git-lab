Que es el CSS?

CSS es un lenguaje que se usa para definir el diseño y la apariencia de una pagina web

Que son los selectores?

son patrones que se utilizan para seleccionar y aplicar estilos a elementos HTML. Sirven para indicar qué elementos del documento recibirán ciertas reglas de diseño.

Propiedades

son las características que se pueden modificar en un elemento HTML para cambiar su apariencia o comportamiento.

Ejemplos

  • Color:Cambia el color del texto.
  • font-size:Cambia el tamaño de la fuenta.
  • font-weight:Hace el texto mas grueso.
  • text-aling:Alinea el texto.
  • text-decoration:Le pone cosas al texto.

Valores

Los valores en CSS son los datos que se asignan a una propiedad para definir su apariencia o comportamiento. Cada propiedad en CSS tiene uno o más valores posibles que determinan cómo se aplicará el estilo al elemento HTML.

Diferencia entre Selectores

  • Por tipo:Se usa para seleccionar todos los elementos de un tipo especifico.
  • Por clase: Se usa para seleccionar elementos que tienen una clase específica..
  • Por identificador:Se utiliza para seleccionar un único elemento con un identificador específico.
