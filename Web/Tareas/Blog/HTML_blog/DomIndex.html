Preguntas

  1. ¿Qué es el DOM?

    2. El DOM (Document Object Model) es una estructura de árbol que representa todos los elementos, etiquetas y contenido de una página web, permitiendo que los lenguajes de programación (como JavaScript) puedan acceder, modificar y manipular esos elementos de forma dinámica.

    Referencia

  2. ¿Qué es un nodo?

    3. Un nodo en el contexto del DOM es cada una de las piezas que forman la estructura del documento. Básicamente, todo lo que está en el HTML es un nodo: etiquetas, texto, atributos, comentarios, etc.

    Referencia

  3. ¿Cómo se accede a un elemento del DOM?

    4. Se accede utilizando métodos como getElementById, getElementsByClassName, getElementsByTagName o querySelector, entre otros. Por ejemplo: document.getElementById("miElemento").

    Referencia

  4. ¿Cómo se modifica un elemento del DOM?

    5. Se puede modificar usando sus propiedades, como textContent, innerHTML, o sus atributos. Por ejemplo: document.getElementById("miElemento").textContent = "Nuevo texto".

    Referencia

  5. ¿Cuál es la diferencia entre getElementby* y querySelector*?

    6. getElementBy* busca elementos por ID, clase o etiqueta y devuelve un objeto o lista HTMLCollection. querySelector* usa selectores CSS y devuelve un solo elemento o una NodeList. Ejemplo: document.getElementById("id") vs document.querySelector("#id").

    Referencia

  6. ¿Qué es un evento?

    7. Un evento es cualquier interacción o acción del usuario con la página, como un clic, movimiento de mouse, o escribir en un input.

    Referencia

  7. ¿Qué es un listener?

    8. Es una función que "escucha" un evento y ejecuta un código cuando ese evento ocurre. Por ejemplo: element.addEventListener("click", funcion).

    Referencia

  8. ¿Qué es un callback?

    9. Es una función que se pasa como argumento a otra función y se ejecuta después de que ocurra un evento o una acción específica.

    Referencia

  9. ¿Qué es un objeto de evento?

    10. Es un objeto que contiene información sobre el evento que ocurrió, como el tipo de evento, el elemento objetivo y las coordenadas del mouse, entre otros.

    Referencia
